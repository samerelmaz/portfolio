Future Brain

A face-recognition web app that smartly detects a person's face and draws a bounding box around it. Built using HTML, CSS and React for developing the single-page front-end. On the back-end, an Express-powered server runs on Node and integrates an external API for detecting faces. A PostgreSQL database is used to securely store user's login information and images submitted. Passwords are encrypted using bcrypt. Finally, it was deployed to Heroku.

Notekeeper

A note-keeping web app with the ability to create, edit and delete notes. Similarly to Future Brain, this web app was built using HTML, CSS and React for front-end and Node, Express and PostgreSQL for the back-end. CRUD operations are performed on the database to create, read, update and delete notes, as well as to register and log-in users. Passwords are encrypted using bcrypt. Also deployed to Heroku.
